Instruction:
  • Enter needed fields in Setting tab
  • Enter observation data in Input tab;
    each row can contain a value in the following format:
    • Time in format HHMM
    • For field of obstruction enter a percent (e.g. 20%)
    • LM (Limited magnitude) (e.g Lm=6.1)
    • Dec (Declination in degrees) (e.g Dec=35)
    • RA (Right Ascension in degrees) (e.g RA=275)
    • Meteor magnitude from -6 until 7
      • For sporadic add minus at end (e.g 3-)
      • For minor shower add at end a meteor code of the shower (e.g.: 3CAP)
        • from list of minor showers in settings: {{settings.showers}}
    • For observation break enter two slashes (//)
  • Download the calculated tabs in csv format as following:
    select Count Distribution tab, export as CSV;
    select Magnitude Distribution tab, export as CSV;
  • Navigate to IMO upload observation and follow instraction to upload the two downloaded csv files
